jQuery wprowadzenie

jQuery - to biblioteka a raczej zbiór funkcji, które ułatwiają korzystanie z JavaScript (ułatwiają jego interpretację). jQuery można też nazwać frameworkiem, który działa podobnie jak np. Bootstrap (CSS). Mówiąc krótko ułatwia i przyspiesza korzystanie z natywnych własciwosci JS.

Jeżeli tylko potrafimy stosować selektory css, to bez problemu będziemy w stanie pokierować jQuery w dokładnie takie miejsce jakie chcemy

Bibliotekę, z powodzeniem możemy sciągnąć z oficjalnej strony

( https://code.jquery.com/ )

W celu instalacji biblioteki na stronie wystarczy dodać pobrany plik między znacznikami <body>...</body>

Następnie między znacznikami <script>...</script> stosujemy predefiniowany znacznik kodu, ktory umożliwi nam odwołanie się do elementow drzewa DOM po załadowaniu zawartosci strony. Zgodnie z poniższym snipetem

    
    //wersja jQuery
    $(document).ready(function() {
        $( "p" ).last().css( "background-color", "red" );
    });

    //wersja JavaScript
    let paragraph = document.querySelector("p:last-of-type");
    paragraph.style.background = "red";

    //wersja 2 - ta sama funkcjonalnosć, lecz oszczędniejsza w zapisie ;)
    $(function() {
        $("#jquery").text().toUpperCase()
    });

    //wersja 2 JavaScript
    let elem = document.querySelector("#jquery");
    elem.style.textTransform = "upperCase";

    //wersja 3 - wykona się po całkowitym załadowaniu strony
    $(window).on("load", function() {
        console.log("Strona w pełni załadowana, zawsze odpali się pozniej niż document ready");
   });

Konstrukcję zapytania możemy rozłożyć na dwie częsci, selektor oraz funkcję jaką chcemy wykonać na zaznaczonym elemencie. Funkcję, można ze sobą łączyć. 

    
    $('element').akcja('parametry');

$() – tak naprawdę wszystko co robimy w tym frameworku, kręci się wokoło tego aliasu do funkcji o nazwie jQuery(). To ona tworzy obiekt jQuery.

element – jak już wczesniej wspomniałem wybieranie elementów jest oparte o selektory CSS. Dostępny jest ogromny wybór selektorów ze specyfikacji CSS oraz kilkanaście autorskich rozwiązań autorów jQuery.

akcja – tu umieszczamy akcje, która będzie wykonywana na danym elemencie.

parametry – jak sama nazwa wskazuje, wpisujemy tu parametry danej akcji.

Przykład zastosowania jQuery przy wspołpracy z komponentem Bootstrap: 
    
    //Dla przykładu przechwycimy event "show.bs.collapse", ktory pojawia się podczas kliknięcia w przycisk
    $(function() {
        //wywołamy akcję show na przycisku
        $("#myCollapsedButton").on('click', () => {
            $("#collapsePlace").collapse('toggle');
        })
    
        //rejestrujemy akcję i sprzedajemy consol loga o zaistniałym fakcie
        $("#collapsePlace").on('show.bs.collapse', function () {
            console.log("I was touched :)");
        }); 
    });

JavaScript przykłady kodu zaimplementowane na stronie

Jako pierwszemu przyglądniemy się smoothScrolling


//$('.nav a').on('click', ... : ustawia EventListener na odnosnikach menu z klasą nav
$('.nav a').on('click', (e) => {
    //argument funkcji "e" daje nam dostęp do wielu informacji o klikniętym elemencie
    //stała hash, przechowuje atrybut "hash" klikniętego linku
    const hash = e.currentTarget.hash;

    //sprawdzamy czy kliknięty odnosnik, posiada jakikolwiek hash 
    if(hash) {
        //zapinamy metodę animacji na bloku html
        $('html, body').animate({
            //zamieniamy stałą "hash" na obiekt jQuery $(hash),
            //zapinamy się z metodą .offset(), która zbiera informację o pozycji elementu obiekt {top: xx, left: xx}
            scrollTop: $(hash).offset().top
            //jako następny parametr funkcji animate, przekazujemy czas trwania animacji
        }, 800, () => {
            //po zakończeniu animacji, dodajemy do adresu url kliknięty hash
            //przykład: http://127.0.0.1:5500/pages/modul2/modul2.html#jqueryPrzyklady
            location.hash = hash
        })
    }
});

Zastosowany CSS


#goUp {
    position: fixed;
    bottom: 30px;
    right: 30px;
    visibility: hidden;
    cursor: pointer;
}

html


<div id="goUp">
    <i class="fas fa-angle-double-up fa-3x"></i>
</div>

jQuery


//pierwsza częsć odkrywa lub ukrywa element zawierający strzałki
$(window).on('scroll', () => {
    if($('#goUp').offset().top >= 1500) {
        $('#goUp').css('visibility', 'visible');
    } else {
        $('#goUp').css('visibility', 'hidden');
    }
});

//dodaje animację podobnie jak w przypadku navigacji
$('#goUp').on('click', () => {
    $('html, body').animate({
        scrollTop: 0
    }, 800);
});
